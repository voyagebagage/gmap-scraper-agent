# Research Workflow

This workflow guides you through researching a topic using the web search tool.

## Steps

1. **Understand the Topic**
   - Goal: Identify the core keywords.

2. **Perform Initial Search**
   - Action: Run `python3 tools/web_search.py --query "[Topic Keywords]"`
   - Output: JSON list of search results.

3. **Analyze Results**
   - Read the snippets from the search results.
   - If information is insufficient, refine the query and repeat Step 2.

4. **Summarize Findings**
   - Create a brief summary of the findings in `.tmp/research_summary.txt`.
